# **HuobiAutomaticTradePlatform**

## **火币自动化交易平台机器人**

### **一.功能设计**
1. 大单买卖监测（交易量在10+BTC的买卖监测）
2. 异常波动监测（一分钟内3%+—，三分钟内5%+-，etc...）
3. 支持涨跌幅度超过限度时邮件通知
4. 币安与火币的搬砖功能
5. 上币监测以及限价买入
6. 通过计算N分钟内M的货币的平均变化趋势，判断火币网大盘的涨跌
7. 法币交易识别超低订单,低价秒杀
8. 设置收益点，止损点，到点自动交易
9. 添加测试脚本模拟买入/卖出，便于根据结果修改参数
